Introduction

JavaScript is a versatile programming language used for web development.

It allows you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

console.log('Hello, world!');
Basics

The basics of JavaScript include variables, data types, and basic operators.

Variables can be declared using var, let, or const.

let x = 5;
Functions

Functions are reusable blocks of code that perform a specific task.

They can take parameters and return a value.

function greet(name) { return 'Hello, ' + name; }
Objects

Objects are collections of related data and functionality.

They consist of properties and methods.

let person = { name: 'John', age: 30 };
Arrays

Arrays are ordered lists of values.

They can hold any type of data and have various methods for manipulating their contents.

let numbers = [1, 2, 3, 4, 5];
